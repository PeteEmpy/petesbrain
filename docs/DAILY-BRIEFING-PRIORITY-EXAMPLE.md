## Key Features

### 1. **Priority-First Sorting**
- Tasks sorted by priority (P0 â†’ P1 â†’ P2 â†’ P3) within each date section
- P0 (Critical/Urgent) tasks always appear first
- Helps you focus on what matters most

### 2. **Time Estimates Displayed**
- Extracted from task notes automatically
- Shows "30 mins", "1-2 hours", etc.
- Total time calculation for daily planning

### 3. **Clickable Task Links**
- Every task title is a clickable link
- Opens task in Google Tasks web interface
- Access full notes and details instantly

### 4. **Visual Priority Indicators**
- **P0**: Critical (URGENT/CRITICAL in notes)
- **P1**: High priority (HIGH/THIS WEEK in notes)
- **P2**: Medium priority (MEDIUM or default)
- **P3**: Low priority (LOW in notes)

### 5. **Smart Capacity Planning**
- "Total estimated time: 1h 55m"
- Helps you plan your day realistically
- Shows if you're overcommitted
