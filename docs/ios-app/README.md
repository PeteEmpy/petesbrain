### Troubleshooting

**Files not syncing?**
```bash
# Check iCloud status
brctl status

# Force sync
killall bird

# Verify folders
ls ~/Library/Mobile\ Documents/com~apple~CloudDocs/PetesBrain-Inbox/
```

**Inbox not processing?**
```bash
# Run manually
python3 ~/Documents/PetesBrain/agents/system/inbox-processor.py

# Check logs
cat ~/.petesbrain-inbox-processor.log
```

**Shortcut errors?**
- Check iCloud Drive permission in Settings
- Verify folder path is correct
- Re-select save location in shortcut
- Check Shortcuts app has Files access

### Common Questions

**Q: Does this work offline?**  
A: Yes! Notes are saved locally on iPhone, then sync when connection returns.

**Q: How fast is the sync?**  
A: Usually 10-30 seconds. Sometimes up to 2 minutes on slow connections.

**Q: Can multiple people use this?**  
A: Yes, if they have access to the same iCloud folder or use shared Mac.

**Q: Do I need a developer account?**  
A: No, not for shortcuts solution. Only needed for native app.

**Q: What if I want Android support?**  
A: Use Email Integration or Telegram Bot option (see IOS-INBOX-APP.md).

**Q: Can I customize the shortcuts?**  
A: Absolutely! Edit them to fit your workflow.

**Q: Should I build a native app?**  
A: Try shortcuts first for 1-2 months, then decide if you need more.
