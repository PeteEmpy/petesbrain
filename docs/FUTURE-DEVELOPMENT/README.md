# Future Development

This directory contains documentation for features and systems that are partially implemented or planned for future development.

## Adding New Future Development Items

When documenting future development:

1. Create a markdown file in this directory
2. Include:
   - Current status (what's built, what's not)
   - Limitations and blockers
   - Future development tasks (organized by phase/priority)
   - Decision points needed
   - Related systems
   - When to revisit

3. Link from main documentation if relevant
