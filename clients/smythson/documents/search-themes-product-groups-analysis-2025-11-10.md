## Listing Groups (Product Groups) Analysis

### Current State: NO Custom Listing Groups

**All asset groups show:**
```
üì¶ Listing Groups: None (all products)
```

**What this means:**
- Every asset group is eligible to show ALL products from the Merchant Center feed
- No filtering by custom_label_0, custom_label_1, category, brand, etc.
- The supplemental feeds you created (Travel Bags, Card Holders, Jewellery Boxes) are NOT being used for filtering

**Potential Issue:**
- "Card Holders" asset group can show travel bags, notebooks, diaries, etc.
- "Notebooks" asset group can show card holders, jewellery boxes, etc.
- Only the search themes and landing page URL guide the asset group focus

**Recommendation:**
Consider adding listing group filters to ensure:
- "Card Holders" asset group ONLY shows products with custom_label_0 = "Card Holders"
- "Travel Bags" asset group ONLY shows products with custom_label_0 = "Travel Bags"
- "Jewellery Boxes" asset group ONLY shows products with custom_label_0 = "Jewellery Boxes"
- Etc.

This would make asset groups more focused and prevent irrelevant product showing for specific search themes.

## Summary of Search Theme Strategy

### Main H&S Campaign:
**Purpose:** Year-round product discovery
**Search themes:** Product-focused, category-specific, some generic gifting
**Landing pages:** Category pages (e.g., /card-holders, /notebooks, /travel-bags)

### Christmas Gifting Campaign:
**Purpose:** Seasonal Christmas gifting (Oct-Dec)
**Search themes:** Christmas-specific gifting themes + gender-specific
**Landing pages:** Gifting pages (/gifts/gifts-for-her, /gifts/gifts-for-him, /christmas.html)

### Key Insight:
The search themes are HEAVILY populated (49-50 per asset group in most cases). This is excellent for giving Google's algorithm strong signals about what searches should trigger each asset group.

However, the LACK of custom listing group filters means products aren't actually restricted by asset group - Google can show any product for any search theme based on its own logic.

## Recommendations

### 1. Add Custom Listing Group Filters
**Priority:** HIGH

For asset groups with supplemental feeds (Card Holders, Travel Bags, Jewellery Boxes), add listing group filters:

**Example - Card Holders Asset Group:**
```
Listing Group Filter:
- custom_label_0 = "Card Holders"
```

This ensures ONLY card holder products show in this asset group.

**Why:** Currently, Google could show notebooks or travel bags when someone searches "luxury card holder" in the Card Holders asset group, which would be suboptimal.

---

### 2. Review "Heroes & Sidekicks" Search Themes
**Priority:** MEDIUM

The generic "Heroes & Sidekicks" asset group still has gifting themes like:
- "luxury gifts for her"

- "luxury christmas gifts for him"
- "personalised luxury gifts for him/her"

**Consider:**
- Remove Christmas-specific themes from this asset group
