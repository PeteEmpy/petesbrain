# Tree2MyDoor

**Website**: https://tree2mydoor.com/

## AI Discoverability Files

### llms.txt
Factual overview of Tree2MyDoor for AI assistant understanding.
- Company information and value proposition
- Products and services
- Target audience and use cases
- Key differentiators

**Status**: Ready for website implementation
**Created**: 2025-10-28

### agents.txt
Comprehensive Q&A guide helping AI assistants answer customer questions about Tree2MyDoor.
- 20 common customer questions with answers
- User workflows
- Key terminology
- Troubleshooting guidance

**Status**: Ready for website implementation
**Created**: 2025-10-28

### Implementation
Both files should be uploaded to the website root directory:
- https://tree2mydoor.com/llms.txt
- https://tree2mydoor.com/agents.txt

This makes Tree2MyDoor discoverable by AI assistants like ChatGPT, Claude, and Gemini.
