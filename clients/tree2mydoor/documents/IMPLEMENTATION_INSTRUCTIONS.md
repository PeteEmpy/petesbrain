## üìä What Happens After Implementation

### **Immediate (Day 1)**
- New assets appear in asset groups
- Status: **PENDING** (Google reviewing)
- Old assets remain active

### **Days 2-7: Learning Phase**
- New assets status: **LEARNING**
- Google algorithm tests new vs old assets
- Both sets actively rotate in auctions

### **Days 8-14: Performance Evaluation**
- Assets get performance labels:
  - **BEST** - Top performers
  - **GOOD** - Solid performers
  - **LOW** - Underperformers
- Google automatically serves better performers more

### **Day 15+: Optimisation**
- Remove old assets that are now labeled **LOW**
- Keep all assets labeled **GOOD** or **BEST** (new and old)
- Google's algorithm naturally phases out poor performers
