# Client Management

This directory contains organized information for all clients, including documents, correspondence, and project materials.

## Structure

Each client folder follows this standard structure:

```
[client-name]/
├── README.md          # Client profile, contacts, and project summary
├── emails/            # Email correspondence with the client
├── documents/         # Google Docs, Word docs, PDFs, general documents
├── spreadsheets/      # Excel files, Google Sheets, CSVs, data files
├── presentations/     # Slides, PowerPoints, Keynote files
├── meeting-notes/     # Notes from meetings, calls, and discussions
└── briefs/            # Project briefs, SOWs, contracts, proposals
```

## Adding a New Client

1. Copy the `_templates/` folder structure
2. Rename to client name (use lowercase-with-dashes format)
3. Create a README.md with client details (see template below)
4. Add client to the list above
