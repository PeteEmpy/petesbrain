## Playbook Structure

Every playbook follows this format:
1. **When to Use** - Scenarios and triggers
2. **Prerequisites** - What you need before starting
3. **Process** - Step-by-step instructions
4. **Interpretation Guide** - How to make decisions
5. **Example** - Real client case (anonymised if needed)
6. **Expected Outcomes** - What to expect
7. **Related Playbooks** - Complementary techniques
