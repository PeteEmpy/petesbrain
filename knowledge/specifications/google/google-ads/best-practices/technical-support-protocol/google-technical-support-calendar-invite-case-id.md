## Full Content

- Maintain case ID reference in all related communications for faster Google support responses
- Use the organizer's calendar invite edit function to reschedule/cancel or add additional attendees
- For Gmail/Google Calendar users: Click 'Yes' in the invite header to confirm attendance
- For non-Google calendar users: Download the ICS file and upload to your calendar system, then confirm via footer link
- If Google representative declines (out of office), contact their manager immediately using provided contact information
- Verify all required attendees have appropriate admin access levels before the scheduled time
- Use phone dial-in as backup if Google Meet connection fails
- Track PARTSTAT values to monitor which attendees have confirmed (NEEDS-ACTION vs ACCEPTED vs DECLINED)
- Escalate to manager contacts (e.g., Ally Hamps - allyhamps@google.com) for urgent matters when primary contact is unavailable
- Respond to calendar invites promptly as appointment will auto-cancel if all attendees decline or don't respond within 24 hours
