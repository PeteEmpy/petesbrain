## Key Insights

- Each website (NDA vs NMA) requires its own unique Google Ads conversion tag ID, even when managed by the same agency - proper account separation is critical
- Enhanced Conversions for Leads requires consistent daily data uploads to function properly; 7+ day gaps in uploads will trigger monitoring requirements
- When implementing conversion tracking across multiple client sites, validate tag IDs per account to avoid cross-contamination and tracking errors
- Google Support requires separate tickets per Ads account, so multi-property implementations need coordinated case management with account managers
