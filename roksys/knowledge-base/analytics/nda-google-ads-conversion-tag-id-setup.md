## Summary

- Peter Empson provided the Google Ads conversion tag ID (AW-1069535771) for National Design Academy (NDA) website

- NMA Enhanced Conversions For Leads is working correctly but requires resumed daily data uploads (none in last 7 days)
- Each institution needs its own AW ID from Tools > Data manager > Google Tag for proper conversion tracking
- Account manager validation required for new conversion tracking implementation on NDA site

## Key Insights

- Multiple brands under same management need separate Google Ads accounts and conversion tracking - cannot share AW IDs across different domains/properties
- Enhanced Conversions for Leads requires consistent daily data uploads to function properly; 7+ days of inactivity indicates a broken integration
- Proper conversion tag implementation requires: correct AW ID retrieval from Ads account > implementation on site > account manager validation > 7-day monitoring period
- Google Support operates on one Ads account per ticket basis - multi-property issues require separate support cases
