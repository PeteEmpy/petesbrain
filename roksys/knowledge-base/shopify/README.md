## Quick Reference

### Essential Shopify Stats (2025)
- **Cart abandonment rate**: 69.82% average
- **Average conversion rate**: 1.5-2.5%
- **Optimized stores**: 3-4%+ conversion rate
- **Mobile traffic**: 50%+ of ecommerce traffic
- **Mobile spending**: $621B estimated (2024)

### Key Integration Points
1. **Google Ads**: Via Google & YouTube app (recommended)
2. **Facebook/Instagram**: Via Meta app
3. **Email Marketing**: Klaviyo, Omnisend, Privy
4. **Analytics**: Google Analytics 4, Shopify Analytics
5. **Feed Management**: FeedOps, AdNabu, DataFeedWatch

### Common Use Cases

**Setting up Google Shopping**:
1. Read: [Google Shopping Master Guide](marketing-channels/google-shopping-master-guide-2025.md)
2. Read: [Product Feed Optimization](product-feeds/google-shopping-feed-optimization-2025.md)
3. Read: [Conversion Tracking Setup](analytics-tracking/conversion-tracking-setup-guide-2025.md)

**Improving Conversion Rate**:
1. Read: [Checkout Optimization](checkout-conversion/checkout-optimization-best-practices-2025.md)
2. Implement guest checkout
3. Add multiple payment options
4. Optimize for mobile
5. Set up cart abandonment emails

**Troubleshooting Conversion Tracking**:
1. Read: [Conversion Tracking Guide](analytics-tracking/conversion-tracking-setup-guide-2025.md)
2. Check if conversion action is enabled
3. Verify Google Ads account ID
4. Test with Google Tag Assistant
5. Allow 24-48 hours for data

## Best Practices by Client Type

### Small Merchants (<¬£100k annual revenue)
**Focus**:
- Start with Google & YouTube app (simplest setup)
- Enable guest checkout immediately
- Optimize product titles and images
- Set up cart abandonment emails
- Use Shopify native features before adding apps

### Medium Merchants (¬£100k-¬£1M annual revenue)
**Focus**:
- Advanced Google Shopping segmentation
- Smart Bidding strategies
- Custom labels for campaign optimization
- Comprehensive checkout optimization
- Email marketing automation (Klaviyo/Omnisend)
- Feed management app for multi-channel

### Large Merchants (¬£1M+ annual revenue)
**Focus**:
- Enterprise feed management
- Performance Max campaigns
- Advanced attribution modeling
- Custom pixel implementation
- Dedicated analytics stack
- Personalization and dynamic content
- International expansion
